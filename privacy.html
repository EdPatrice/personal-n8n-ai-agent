Privacy Policy

Last updated: December 18, 2025

This Privacy Policy describes how this application ("the App") accesses and uses Google user data.

1. Overview

The App is a personal automation assistant built on a self-hosted n8n instance.
It connects to Google services only after explicit user authorization and performs actions strictly requested by the user through conversational commands.

The App is intended for personal use by the account owner and is not a public consumer product.

2. Google Data Accessed

With the user's consent, the App may access the following Google services:

Gmail

  • Read email messages and metadata
  • Send emails and replies
  • Create email drafts
  • Apply labels to messages

Purpose:
To help the user read, summarize, draft, send, reply to, and organize emails on request.

Google Calendar

  • Read calendar events
  • Create, update, and delete calendar events

Purpose:
To manage events, schedules, and reminders as explicitly requested by the user.

Google Tasks

  • Read tasks and reminders
  • Create new tasks

Purpose:
To help the user manage to-dos and reminders.

Google Contacts

  • Read contact information such as names, email addresses, phone numbers, birthdays, and relationships

Purpose:
To identify contacts and retrieve contact details when required to complete user-requested actions (for example, sending an email or creating a reminder).

3. How Data Is Used

Google user data is:

  • Accessed only in response to explicit user requests
  • Used solely to perform the requested action
  • Not used for advertising, profiling, or analytics

The App does not perform background monitoring or automated actions without user initiation.

4. Data Storage and Retention

  • OAuth tokens are stored securely on a self-hosted n8n server.
  • Email content, calendar data, tasks, and contacts are processed transiently during workflow execution.
  • The App does not maintain a separate long-term database of Gmail messages, calendar events, tasks, or contacts.

5. Data Sharing

  • No Google user data is shared with third parties.
  • Data is not sold, rented, or transferred to external services outside the execution of the automation workflows.

6. User Control and Revocation

The user can revoke the App's access at any time by visiting:
https://myaccount.google.com/permissions

Once access is revoked, the App will no longer be able to access Google data.

7. Security

Reasonable technical measures are in place to protect OAuth credentials and ensure that data access is limited to authorized workflows.
